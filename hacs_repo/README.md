# Yija Switch panel

This package is structured for use as a Home Assistant HACS custom repository.

## Repository structure

- `custom_components/yija_switch_panel/` — integration files from the provided version
- `hacs.json` — HACS metadata

## Install with HACS custom repository

1. In HACS, open **Integrations**.
2. Open the menu in the top right and choose **Custom repositories**.
3. Add this repository URL.
4. Select **Integration** as the category.
5. Install **Yija Switch panel**.
6. Restart Home Assistant.

## Manual install

Copy `custom_components/yija_switch_panel` to your Home Assistant `config/custom_components/` directory, then restart Home Assistant.
